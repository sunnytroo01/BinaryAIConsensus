Full state feedback
 Ara Eve Leo Sal 
1x
 Full state feedback is a core method in modern control theory for linear time-invariant systems, where the control input is formed as a linear combination of all measurable state variables to shape the system's closed-loop dynamics. This technique allows engineers to arbitrarily assign the locations of the closed-loop poles in the complex plane, enabling precise control over stability, transient response, and steady-state behavior, provided the system satisfies the controllability condition.
For a system modeled in state-space form as x=Ax+Bu\dot{x} = Ax + Bux=Ax+Bu, with state vector xRnx \in \mathbb{R}^nxRn and input uRmu \in \mathbb{R}^muRm, the full state feedback law is typically u=Kx+vu = -Kx + vu=Kx+v, where KRmnK \in \mathbb{R}^{m \times n}KRmn is the feedback gain matrix and vvv is a reference input. Substituting this law yields the closed-loop system x=(ABK)x+Bv\dot{x} = (A - BK)x + Bvx=(ABK)x+Bv, whose eigenvalues (poles) can be placed at any desired values if the controllability matrix has full rank.
Key design approaches include direct pole placement using Ackermann's formula for single-input systems, which computes KKK based on the desired characteristic polynomial, and the linear quadratic regulator (LQR) method, which optimizes KKK by minimizing a quadratic cost functional subject to the system dynamics via the algebraic Riccati equation. These methods apply to both single-input single-output (SISO) and multi-input multi-output (MIMO) systems, offering advantages over classical control techniques by addressing multivariable interactions holistically.
In practice, full state feedback requires access to the entire state vector; if states are not directly measurable, a state observer (such as a Luenberger observer) reconstructs them, and the separation principle guarantees that the controller and observer can be designed independently without affecting overall stability. This framework emerged in the mid-1960s as part of the shift to state-space methods, with foundational results on pole assignment for controllable multi-input systems established by W. Murray Wonham in 1967.

Background Concepts

State-Space Representation

The state-space representation models linear time-invariant (LTI) dynamical systems using a set of first-order vector differential equations that describe the evolution of internal states in response to inputs. This framework is foundational for full state feedback control, as it explicitly captures the system's dynamics through state variables.
For a continuous-time LTI system, the state-space equations are given by

The components of the state vector x(t)x(t)x(t) represent a minimal set of variables sufficient to describe the complete internal condition of the system at any time, incorporating both the dynamics and the effects of initial conditions to predict future outputs given the inputs. Intuitively, these state variables encode the essential information from the system's past history needed to determine its response to future inputs, ensuring no redundancy in the model.
In contrast to transfer function representations, which describe input-output relationships via rational functions and are straightforward for single-input single-output (SISO) systems but extend to matrix forms for multi-input multi-output (MIMO) cases with increased complexity, the state-space form handles MIMO systems seamlessly through matrix equations that reveal internal state interactions. This structure facilitates analysis of multivariable dynamics without losing insight into the system's order or realizability.
The state-space representation emerged in the 1960s as a cornerstone of modern control theory, with Rudolf E. Kalman playing a pivotal role in its formalization through his development of time-domain methods for filtering, prediction, and system description. Kalman's contributions shifted emphasis from classical frequency-domain techniques to algebraic state-based approaches, enabling rigorous treatments of controllability and optimal design in complex systems.

Controllability

In linear control systems described by the state-space model x=Ax+Bu\dot{x} = Ax + Bux=Ax+Bu, controllability refers to the capability to drive the state vector xxx from any arbitrary initial condition x(0)x(0)x(0) to any specified final state x(tf)x(t_f)x(tf) in finite time tf>0t_f > 0tf>0 through an appropriate choice of the input u(t)u(t)u(t). This property is fundamental for ensuring that full state feedback can achieve desired system behavior, as it guarantees the existence of inputs that can manipulate the entire state space.
The standard test for controllability involves constructing the controllability matrix C=\mathcal{C} = C=, where nnn is the dimension of the state vector and AAA, BBB are the system matrices. The linear system is controllable if and only if C\mathcal{C}C has full rank equal to nnn, meaning its column space spans the entire Rn\mathbb{R}^nRn. This rank condition, known as the Kalman rank condition, provides an algebraic criterion to verify controllability without simulating system trajectories.
A proof of this condition relies on the Cayley-Hamilton theorem, which states that the matrix AAA satisfies its own characteristic equation, allowing higher powers of AAA (beyond An1A^{n-1}An1) to be expressed as linear combinations of lower powers I,A,,An1I, A, \dots, A^{n-1}I,A,,An1. The reachable subspace from the origin under input uuu is the span of the columns of C(t,0)\mathcal{C} \Phi(t,0)C(t,0), where \Phi is the state transition matrix; by Cayley-Hamilton, this span is identical to that of C\mathcal{C}C, so full rank nnn ensures the entire state space is reachable, and thus controllable.
For single-input systems (where BBB is an n1n \times 1n1 vector), C\mathcal{C}C forms an nnn \times nnn square matrix, and controllability requires det(C)0\det(\mathcal{C}) \neq 0det(C)=0. In multi-input systems ( BBB is nmn \times mnm with m>1m > 1m>1), C\mathcal{C}C is nnmn \times nmnnm, and the rank condition checks if rank(C)=n\operatorname{rank}(\mathcal{C}) = nrank(C)=n, which can be computed via singular value decomposition or row echelon form to confirm the column rank.
If the rank of C\mathcal{C}C is less than nnn, the system has uncontrollable modes corresponding to the kernel of the controllability Gramian or the uncontrollable subspace, which cannot be influenced by the input uuu; in such cases, full state feedback can only stabilize or shape the controllable subsystem, leaving uncontrollable dynamics unaffected.

Mathematical Formulation

Feedback Control Law

In full state feedback control, the control input $ u $ for a linear time-invariant system is defined as a linear combination of the state variables, given by the law $ u = -Kx + v $, where $ x \in \mathbb{R}^n $ is the state vector, $ K \in \mathbb{R}^{m \times n} $ is the feedback gain matrix to be designed, and $ v $ is a reference input. The negative sign in the formulation conventionally implements negative feedback, which is essential for stabilizing unstable open-loop dynamics by counteracting deviations in the state from equilibrium. The elements of $ K $ are selected to shape the transient and steady-state response of the closed-loop system, such as adjusting damping, rise time, or overshoot, while ensuring asymptotic stability when the open-loop system is controllable.
This control law applies generally to multi-input multi-output (MIMO) systems, where there are $ m $ control inputs and $ n $ state variables, making $ K $ an $ m \times n $ matrix that maps the full state information to the input space. For single-input single-output (SISO) systems, $ K $ reduces to a row vector, but the principle remains the same: all states contribute to the control action. The formulation assumes perfect knowledge of the entire state vector $ x $, with direct measurement of all components and no sensor noise or modeling uncertainties, which simplifies analysis but may require state estimation in practical scenarios.
In contrast to output feedback, where the control input depends only on the measured outputs $ y = Cx $ with $ C $ being the output matrix (thus limiting information to a subset of states), full state feedback utilizes the complete state to achieve greater flexibility in performance specification, provided the system is controllable to guarantee the existence of a stabilizing $ K $.

Closed-Loop System Dynamics

In full state feedback control, the closed-loop system dynamics are obtained by substituting the control law $ u = -Kx + v $ into the open-loop state-space model $ \dot{x} = Ax + Bu $, $ y = Cx $, yielding the modified state equation $ \dot{x} = (A - BK)x + Bv $ and the output equation $ y = Cx $. This transformation alters the system's natural behavior by incorporating the feedback gain matrix $ K $, which influences the evolution of the state vector $ x $.
The eigenvalues of the closed-loop matrix $ A - BK $, known as the closed-loop poles, determine the system's stability and response characteristics. If the original system is controllable, the gain $ K $ can be selected to assign these poles to any desired locations in the complex plane, thereby shaping the dynamic response. For instance, placing poles with negative real parts ensures asymptotic stability, while their specific positions allow precise control over the system's transient behavior.
The characteristic equation of the closed-loop system is given by $ \det(sI - (A - BK)) = 0 $, which defines the roots corresponding to the closed-loop poles. This polynomial equation encapsulates the influence of the feedback gain on the system's modal structure, enabling designers to match desired pole configurations by solving for $ K $.
The locations of the closed-loop poles directly impact the transient response of the system, including damping ratios and settling times. For example, poles with larger negative real parts increase the speed of response and reduce settling time, while the imaginary parts introduce oscillatory components that affect damping. In second-order systems, the standard form $ s^2 + 2\zeta \omega_n s + \omega_n^2 = 0 $ illustrates how pole placement adjusts $ \zeta $ (damping) and $ \omega_n $ (natural frequency) to achieve specified performance criteria.

Design Methods

Pole Placement Technique

The pole placement technique is a fundamental method in state-space control design for assigning the eigenvalues (poles) of the closed-loop system matrix ABKA - BKABK to specified locations, thereby achieving desired dynamic performance such as specified settling time and damping characteristics. This approach relies on the full state feedback law u=Kxu = -Kxu=Kx, where KKK is the gain matrix, and is applicable to linear time-invariant systems that are controllable. For a controllable pair (A,B)(A, B)(A,B), the technique guarantees the existence of a KKK that places the closed-loop poles arbitrarily, as established by the pole placement theorem.
The design process begins with determining the desired pole locations based on performance specifications. For instance, in second-order systems, poles are selected to meet a target damping ratio \zeta and natural frequency n\omega_nn, ensuring adequate overshoot and response speed while maintaining stability (real parts negative). It also assumes perfect state measurement and exact plant knowledge, limiting applicability in uncertain environments.

Inverted Pendulum Control

The inverted pendulum on a cart serves as a canonical example of applying full state feedback to stabilize a single-input multivariable system that is inherently unstable. In this setup, a pendulum is hinged to a horizontally moving cart, and the control input is a force applied to the cart to maintain the pendulum in its upright equilibrium position despite disturbances or initial deviations. This benchmark illustrates the challenges of underactuated control, where the single actuator influences multiple degrees of freedom, requiring careful state feedback design to achieve both pendulum balancing and cart positioning.
The dynamics are linearized around the upright position, where the state vector is defined as $ x = ^T $, with $ \theta $ the pendulum angle from the vertical (upright at $ \theta = 0 $), $ \dot{\theta} $ the angular velocity, $ x_c $ the cart position, and $ \dot{x}_c $ the cart velocity. The resulting state-space model is $ \dot{x} = A x + B u $, where $ A $ is a 44 matrix capturing the coupled linear dynamics influenced by gravity and inertia, and $ B $ is a 14 input matrix reflecting the effect of the cart force $ u $. The explicit forms of $ A $ and $ B $ depend on system parameters such as cart mass $ M $, pendulum mass $ m $, pendulum length to center of mass $ l $, and gravitational acceleration $ g $; for instance, the $ A $ matrix includes positive terms in the angular dynamics row that induce instability due to gravity.
Controllability of the linearized model is verified by computing the controllability matrix $ \mathcal{C} = $, which has full rank 4 for physically realistic parameters (e.g., $ M > 0 $, $ m > 0 $, $ l > 0 $), confirming that full state feedback can arbitrarily place the closed-loop poles.
For controller design, the linear quadratic regulator (LQR) is employed to compute the state feedback gain $ K $ by minimizing the quadratic cost $ J = \int_0^\infty (x^T Q x + u^T R u) , dt $, with $ Q = \diag(1, 1, 1, 1) $ weighting all states equally and $ R = 1 $ penalizing control effort. The optimal $ K $ is obtained via the solution $ P $ to the algebraic Riccati equation $ A^T P + P A - P B R^{-1} B^T P + Q = 0 $, yielding $ K = R^{-1} B^T P $. This choice of weights promotes balanced performance, but adjusting the diagonal elements of $ Q $ (e.g., increasing those for $ \theta $ and $ \dot{\theta} $) prioritizes pendulum stabilization over cart positioning, while higher $ R $ reduces aggressive control actions at the potential cost of slower convergence.
In simulations starting from an initial tilt (e.g., $ \theta(0) = 0.1 $ rad), the open-loop system exhibits exponential instability, with the pendulum angle diverging due to the positive eigenvalue in $ A $ associated with the upright mode. Applying the LQR feedback $ u = -K x $ stabilizes the closed-loop system, returning the pendulum to upright while the cart undergoes damped oscillations before settling near zero position.
The inverted pendulum has been a standard benchmark in control and robotics literature since the 1960s.

Implementation Aspects

State Measurement Assumptions

In the theoretical foundation of full state feedback control, it is assumed that all state variables of the dynamical system are directly measurable through sensors, with no measurement delays or noise corrupting the state vector $ x $ used in the feedback law $ u = -Kx $. This idealization enables precise pole placement or optimization without the need for estimation, relying on the system's controllability for arbitrary closed-loop dynamics assignment.
However, practical implementation faces significant challenges, as measuring every state often incurs high sensor costs due to the need for multiple, accurate devices across complex systems, alongside reliability and installation constraints. Certain states, such as internal fluid velocities in pipelines or temperatures within chemical reactors, remain inherently unmeasurable without invasive instrumentation, violating the full accessibility assumption. Moreover, even when measurements are available, sensor noise can be amplified by the feedback gain matrix $ K $, especially for high-gain designs, leading to control input saturation or instability in sensitive applications.
These limitations prompt trade-offs between full state feedback and alternative approaches like reduced-order observers, which estimate only the unmeasurable subset of states to minimize sensor requirements and noise sensitivity while preserving control performance. In scenarios with partial measurements, such observers integrate seamlessly with the feedback law, reducing overall system cost compared to equipping sensors for all states.
Advancements in the 1970s extended full state feedback to noisy environments by integrating it with Kalman filters for optimal state estimation, leveraging the separation principle to decouple estimation from control design and enable robust performance under stochastic disturbances. This combination, rooted in linear quadratic Gaussian (LQG) frameworks, addressed measurement imperfections without altering the core feedback structure.
When the full state measurement assumption cannot be met, control strategies shift to output feedback or full observer-based designs, where estimated states replace direct measurements to ensure stability and tracking.
