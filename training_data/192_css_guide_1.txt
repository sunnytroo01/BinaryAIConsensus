CSS Fundamentals. How to style any element on any web page.

CSS selectors target elements to style. The most important selectors:

Element selector targets all elements of that type. p { color: blue; } makes all paragraphs blue.

Class selector targets elements with a specific class. .card { border: 1px solid #ddd; padding: 20px; border-radius: 8px; } Apply with <div class="card">

ID selector targets one unique element. #hero { background: linear-gradient(135deg, #667eea, #764ba2); } Apply with <div id="hero">

Descendant selector targets elements inside other elements. .nav a { color: white; text-decoration: none; } targets links inside elements with class nav.

Child selector targets direct children only. .menu > li { display: inline-block; } targets only li elements that are direct children of .menu, not nested lis.

Pseudo-classes target elements in specific states. a:hover { color: red; } when mouse hovers. button:active { transform: scale(0.95); } when clicked. input:focus { border-color: blue; } when focused. li:first-child { font-weight: bold; } first item. li:nth-child(even) { background: #f5f5f5; } even rows.

The Box Model. Every element is a box with content, padding, border, and margin.

content is the actual text or image inside.
padding is space between content and border.
border is a line around the padding.
margin is space between this element and neighboring elements.

By default, width and height set only the content size. Padding and border add to the total size. This is confusing. Fix it globally:

*, *::before, *::after { box-sizing: border-box; }

Now width and height include padding and border. This is how every modern website works.

Colors in CSS:

Named colors: color: red; color: tomato; color: cornflowerblue;
Hex codes: color: #ff0000; color: #333; (shorthand for #333333)
RGB: color: rgb(255, 0, 0); color: rgba(0, 0, 0, 0.5); (with transparency)
HSL: color: hsl(0, 100%, 50%); color: hsla(200, 80%, 50%, 0.8);

Typography:

font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
This font stack uses the system font on every platform. Fast, familiar, no font loading.

font-size: 16px; (absolute) or font-size: 1rem; (relative to root)
Use rem for responsive sizing. 1rem equals the root font size, typically 16px.

font-weight: 400; (normal) or font-weight: 700; (bold)
line-height: 1.6; (unitless is best, multiplied by font size)
letter-spacing: 0.5px; (adds space between characters)
text-transform: uppercase; (forces uppercase display)
text-align: center; (left, right, center, justify)

Spacing:

margin: 20px; (all sides)
margin: 20px 40px; (top/bottom, left/right)
margin: 10px 20px 30px 40px; (top, right, bottom, left - clockwise)
margin: 0 auto; (centers a block element horizontally)

padding uses the same syntax as margin.

gap: 20px; (space between flex or grid children - cleaner than margins)

Display:

display: block; (takes full width, starts on new line - div, p, h1)
display: inline; (only as wide as content, no new line - span, a, strong)
display: inline-block; (inline but respects width/height)
display: flex; (enables Flexbox layout)
display: grid; (enables Grid layout)
display: none; (hides element completely)

Position:

position: static; (default, normal flow)
position: relative; (offset from normal position, still occupies original space)
position: absolute; (removed from flow, positioned relative to nearest positioned ancestor)
position: fixed; (removed from flow, positioned relative to viewport, stays on scroll)
position: sticky; (switches from relative to fixed when scroll threshold is reached)

Use top, right, bottom, left to offset positioned elements.

.fixed-header { position: fixed; top: 0; left: 0; right: 0; z-index: 100; }
.sticky-nav { position: sticky; top: 0; }
.tooltip { position: absolute; bottom: 100%; left: 50%; transform: translateX(-50%); }
