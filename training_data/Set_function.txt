Algebraically, set functions link to group structures through invariance under group actions, particularly in locally compact groups where Haar measures serve as canonical examples. A set function \mu defined on subsets of a group GGG is translation-invariant if (A+x)=(A)\mu(A + x) = \mu(A)(A+x)=(A) for all measurable sets AGA \subseteq GAG and all xGx \in GxG, where A+x={a+xaA}A + x = \{a + x \mid a \in A\}A+x={a+xaA}. Extensions and Constructions

From Semirings to Algebras

A semiring of sets is a non-empty collection P\mathcal{P}P of subsets of a set XXX that is closed under finite intersections, meaning that if E,FPE, F \in \mathcal{P}E,FP, then EFPE \cap F \in \mathcal{P}EFP, and such that for any E,FPE, F \in \mathcal{P}E,FP with EFE \subseteq FEF, the difference FEF \setminus EFE can be expressed as a finite disjoint union of sets from P\mathcal{P}P. This structure provides a foundational class for defining set functions, often starting with finitely additive functions on P\mathcal{P}P. Semirings are particularly useful in measure theory because they allow systematic extensions while preserving key properties like monotonicity.
To extend a set function :P\mu: \mathcal{P} \to :P defined on a semiring P\mathcal{P}P to the algebra A\mathcal{A}A generated by P\mathcal{P}P, one defines \mu on A\mathcal{A}A using finite disjoint unions of elements from P\mathcal{P}P. This definition ensures that if \mu is finitely additive and monotone on P\mathcal{P}P, then the extended \mu inherits these properties on A\mathcal{A}A, maintaining finite additivity for disjoint unions in the algebra and monotonicity for nested sets.
The extension is unique provided that the whole space XXX can be covered by a finite disjoint union of sets from P\mathcal{P}P, ensuring that the algebra A\mathcal{A}A includes XXX and that no alternative representations affect the values of \mu. This finite covering condition guarantees that the measure remains well-defined and consistent across equivalent decompositions in A\mathcal{A}A.
