CSS Grid Complete Guide. Two-dimensional layouts made simple.

CSS Grid is a two-dimensional layout system that handles both columns and rows simultaneously. While Flexbox is great for one-dimensional layouts, Grid excels at complex page layouts where you need precise control over both dimensions.

Basic grid setup:
.grid { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 20px; }
This creates a three-column grid with equal-width columns and 20px gaps between items.

fr is the fractional unit. 1fr means one fraction of the available space. 2fr 1fr 1fr creates three columns where the first is twice as wide as the others.

Explicit column sizes:
grid-template-columns: 200px 1fr 300px; (fixed sidebar, flexible content, fixed sidebar)
grid-template-columns: repeat(3, 1fr); (three equal columns)
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)); (as many 250px+ columns as will fit)
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); (same but stretches to fill remaining space)

auto-fill vs auto-fit: auto-fill creates empty columns if there is extra space. auto-fit collapses empty columns so existing items stretch to fill the space. For responsive card grids, auto-fit is usually what you want.

Row sizing:
grid-template-rows: 80px 1fr 60px; (fixed header, flexible content, fixed footer)
grid-auto-rows: minmax(100px, auto); (auto-generated rows are at least 100px tall)

Placing items in specific grid cells:
.item { grid-column: 1 / 3; } (spans from column line 1 to column line 3, covering two columns)
.item { grid-row: 2 / 4; } (spans from row line 2 to row line 4, covering two rows)
.item { grid-column: span 2; } (spans two columns from its natural position)
.item { grid-area: 1 / 1 / 3 / 3; } (row-start / col-start / row-end / col-end)

Named grid areas for readable layouts:
.layout {
  display: grid;
  grid-template-columns: 250px 1fr;
  grid-template-rows: 80px 1fr 60px;
  grid-template-areas:
    "header header"
    "sidebar content"
    "footer footer";
  min-height: 100vh;
}
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.footer { grid-area: footer; }

This creates a classic page layout with a full-width header, a sidebar, a content area, and a full-width footer. The areas are named in the template, making the layout structure immediately readable.

Alignment in Grid:
justify-items: center; (horizontally centers items within their grid cells)
align-items: center; (vertically centers items within their grid cells)
place-items: center; (shorthand for both)
justify-content: center; (centers the entire grid within the container)
align-content: center; (vertically centers the entire grid)
place-content: center; (shorthand for both)

Common Grid patterns:

Responsive card grid:
.cards { display: grid; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); gap: 24px; }

Dashboard layout:
.dashboard { display: grid; grid-template-columns: 250px 1fr 300px; grid-template-rows: 60px 1fr; gap: 16px; height: 100vh; }

Image gallery:
.gallery { display: grid; grid-template-columns: repeat(3, 1fr); gap: 4px; }
.gallery img { width: 100%; height: 100%; object-fit: cover; }
.featured { grid-column: span 2; grid-row: span 2; }

Holy grail layout with media query:
.page { display: grid; grid-template-columns: 1fr; }
@media (min-width: 768px) {
  .page { grid-template-columns: 200px 1fr 200px; }
}

Grid is the most powerful layout tool in CSS. It eliminates the need for float hacks, clearfix tricks, and complex nesting that plagued older CSS layout techniques. Learn Grid and you can build any layout.
