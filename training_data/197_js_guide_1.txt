JavaScript Fundamentals. The complete language reference.

Variables. Use const for values that do not change. Use let for values that do. Never use var.

const name = "Alice";
const age = 30;
const isActive = true;
const scores = [95, 87, 92];
const user = { name: "Alice", age: 30 };

let count = 0;
count = count + 1;

Data types. JavaScript has 7 primitive types and 1 complex type.

String: "hello" or 'hello' or `hello ${name}` (template literal with interpolation)
Number: 42, 3.14, -7, Infinity, NaN
Boolean: true, false
null: intentional empty value
undefined: declared but not assigned
BigInt: 9007199254740991n (for very large integers)
Symbol: Symbol("id") (unique identifier)
Object: { key: "value" } (includes arrays, functions, dates, and more)

Operators. Arithmetic: + - * / % ** (modulo and exponent). Comparison: === !== > < >= <= (always use === not ==). Logical: && || ! (and, or, not). Assignment: = += -= *= /=. Ternary: condition ? valueIfTrue : valueIfFalse.

const status = age >= 18 ? "adult" : "minor";
const greeting = name || "stranger"; (uses "stranger" if name is falsy)
const displayName = user?.name ?? "unknown"; (optional chaining + nullish coalescing)

Strings. Strings are immutable sequences of characters.

const str = "Hello World";
str.length; returns 11
str.toUpperCase(); returns "HELLO WORLD"
str.toLowerCase(); returns "hello world"
str.includes("World"); returns true
str.startsWith("Hello"); returns true
str.indexOf("World"); returns 6
str.slice(0, 5); returns "Hello"
str.split(" "); returns ["Hello", "World"]
str.replace("World", "JavaScript"); returns "Hello JavaScript"
str.trim(); removes whitespace from both ends
str.padStart(15, "-"); returns "----Hello World"

Template literals use backticks and allow multi-line strings and interpolation:
const message = `Hello ${name}, you are ${age} years old.
This is a multi-line string.`;

Arrays. Ordered lists of values.

const fruits = ["apple", "banana", "cherry"];
fruits.length; returns 3
fruits[0]; returns "apple"
fruits.push("date"); adds to end
fruits.pop(); removes from end
fruits.unshift("apricot"); adds to beginning
fruits.shift(); removes from beginning
fruits.includes("banana"); returns true
fruits.indexOf("cherry"); returns 2

Array methods that return new arrays:
fruits.map(f => f.toUpperCase()); transforms each element
fruits.filter(f => f.length > 5); keeps elements matching condition
fruits.find(f => f === "banana"); returns first match
fruits.some(f => f === "apple"); returns true if any match
fruits.every(f => f.length > 3); returns true if all match
fruits.reduce((sum, f) => sum + f.length, 0); reduces to single value
fruits.sort(); sorts alphabetically
fruits.reverse(); reverses order
fruits.slice(1, 3); returns elements 1 and 2 (not 3)
fruits.join(", "); returns "apple, banana, cherry"

Spread operator: const moreFruits = [...fruits, "elderberry"];
Destructuring: const [first, second, ...rest] = fruits;

Objects. Collections of key-value pairs.

const person = { name: "Alice", age: 30, city: "New York", greet() { return `Hi, I am ${this.name}`; } };

person.name; returns "Alice" (dot notation)
person["age"]; returns 30 (bracket notation)
person.email = "alice@example.com"; adds new property
delete person.city; removes property
Object.keys(person); returns array of keys
Object.values(person); returns array of values
Object.entries(person); returns array of [key, value] pairs
"name" in person; returns true

Destructuring: const { name, age, city = "Unknown" } = person;
Spread: const updated = { ...person, age: 31 };

Functions. Three ways to define functions.

Function declaration (hoisted):
function add(a, b) { return a + b; }

Function expression (not hoisted):
const add = function(a, b) { return a + b; };

Arrow function (concise, no own this):
const add = (a, b) => a + b;
const double = n => n * 2; (single param, no parens needed)
const getUser = () => ({ name: "Alice" }); (return object, wrap in parens)

Default parameters: function greet(name = "stranger") { return `Hello ${name}`; }
Rest parameters: function sum(...numbers) { return numbers.reduce((a, b) => a + b, 0); }

Control flow:

if (condition) { } else if (otherCondition) { } else { }

switch (value) { case "a": break; case "b": break; default: break; }

for (let i = 0; i < 10; i++) { }
for (const item of array) { }
for (const [key, value] of Object.entries(obj)) { }
while (condition) { }
do { } while (condition);

try { riskyCode(); } catch (error) { console.error(error.message); } finally { cleanup(); }
