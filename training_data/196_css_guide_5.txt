CSS Modern Techniques and Tricks. Advanced patterns that professional developers use daily.

CSS Custom Properties are variables that cascade and can be changed dynamically with JavaScript. Define them with double dashes and use them with var().

:root { --primary: #3b82f6; --radius: 8px; --shadow: 0 4px 6px rgba(0,0,0,0.1); }
.card { background: white; border-radius: var(--radius); box-shadow: var(--shadow); }
.button { background: var(--primary); border-radius: var(--radius); }

Change a variable with JavaScript: document.documentElement.style.setProperty('--primary', '#ef4444');

Smooth scrolling for the entire page:
html { scroll-behavior: smooth; }
Now anchor links like <a href="#section2"> will scroll smoothly instead of jumping.

Aspect ratio for consistent media containers:
.video-container { aspect-ratio: 16 / 9; }
.square { aspect-ratio: 1; }
.portrait { aspect-ratio: 3 / 4; }

Object-fit controls how images fill their containers:
img { width: 100%; height: 300px; object-fit: cover; }
cover fills the container while maintaining aspect ratio, cropping as needed. contain fits the entire image inside the container, adding letterboxing if needed.

Backdrop-filter creates glass effects:
.glass { background: rgba(255, 255, 255, 0.1); backdrop-filter: blur(10px); border: 1px solid rgba(255, 255, 255, 0.2); }

CSS gradients for backgrounds:
background: linear-gradient(135deg, #667eea, #764ba2); (angled gradient)
background: radial-gradient(circle at center, #667eea, #764ba2); (circular gradient)
background: conic-gradient(red, yellow, green, blue, red); (rainbow wheel)

Text gradient effect:
.gradient-text { background: linear-gradient(135deg, #667eea, #764ba2); -webkit-background-clip: text; -webkit-text-fill-color: transparent; }

Truncate text with ellipsis:
.truncate { white-space: nowrap; overflow: hidden; text-overflow: ellipsis; }

Multi-line text clamp:
.clamp-3 { display: -webkit-box; -webkit-line-clamp: 3; -webkit-box-orient: vertical; overflow: hidden; }

Smooth hover card lift:
.card { transition: transform 0.2s ease, box-shadow 0.2s ease; }
.card:hover { transform: translateY(-4px); box-shadow: 0 12px 24px rgba(0,0,0,0.15); }

CSS counters for automatic numbering:
.steps { counter-reset: step; }
.step::before { counter-increment: step; content: counter(step) ". "; font-weight: bold; color: #3b82f6; }

Scroll snap for carousel-like behavior:
.scroll-container { display: flex; overflow-x: auto; scroll-snap-type: x mandatory; gap: 16px; }
.scroll-item { scroll-snap-align: start; min-width: 300px; flex-shrink: 0; }

Hide scrollbar while keeping scroll functionality:
.container { overflow: auto; scrollbar-width: none; }
.container::-webkit-scrollbar { display: none; }

CSS-only tooltip:
.tooltip { position: relative; }
.tooltip::after { content: attr(data-tip); position: absolute; bottom: 100%; left: 50%; transform: translateX(-50%); background: #333; color: white; padding: 4px 8px; border-radius: 4px; font-size: 12px; white-space: nowrap; opacity: 0; pointer-events: none; transition: opacity 0.2s; }
.tooltip:hover::after { opacity: 1; }
Use with: <span class="tooltip" data-tip="This is helpful info">Hover me</span>

Focus-visible for keyboard-only focus styling:
button:focus-visible { outline: 2px solid #3b82f6; outline-offset: 2px; }
This shows a focus ring only when the user is navigating with a keyboard, not when clicking with a mouse.

Container queries for component-based responsive design:
.card-container { container-type: inline-size; }
@container (min-width: 400px) { .card { display: flex; } }
Container queries respond to the size of the parent container rather than the viewport, making components truly reusable regardless of where they are placed on the page.
