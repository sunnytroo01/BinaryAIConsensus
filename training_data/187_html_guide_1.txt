HTML Elements Reference. Every HTML element explained with examples.

The DOCTYPE declaration tells the browser which version of HTML the page uses. Always start with <!DOCTYPE html> for HTML5.

The html element wraps everything. Add lang="en" for English pages. <html lang="en">

The head contains metadata. Title sets the browser tab text. Meta charset="UTF-8" ensures all characters display correctly. Meta viewport makes pages mobile-friendly.

Example head:
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>My Page Title</title>
<link rel="stylesheet" href="styles.css">
</head>

The body contains all visible content. Everything the user sees goes inside body.

Headings h1 through h6 create titles. h1 is the most important, used once per page for the main title. h2 for sections. h3 for subsections. Never skip levels.

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection Title</h3>

Paragraphs use the p tag. Browsers add automatic spacing above and below.

<p>This is a paragraph. It can contain as much text as needed.</p>

Links use the a tag with href for the URL. Add target="_blank" to open in a new tab. Add rel="noopener noreferrer" for security on external links.

<a href="https://example.com" target="_blank" rel="noopener noreferrer">Visit Example</a>

Images use the img tag. It has no closing tag. Always include alt text for accessibility.

<img src="photo.jpg" alt="A golden retriever playing in the park" width="600" height="400">

Lists come in two types. Unordered lists ul use bullet points. Ordered lists ol use numbers. Each item uses li.

<ul>
<li>First item</li>
<li>Second item</li>
<li>Third item</li>
</ul>

<ol>
<li>Step one</li>
<li>Step two</li>
<li>Step three</li>
</ol>

Divs are generic block containers for grouping content. Spans are inline containers for wrapping text within paragraphs.

<div class="card">
<h2>Card Title</h2>
<p>Card content goes here</p>
</div>

<p>This is <span class="highlight">highlighted</span> text.</p>

Forms collect user input. The form element wraps all inputs. Action specifies where data is sent. Method specifies GET or POST.

<form action="/submit" method="POST">
<label for="name">Name:</label>
<input type="text" id="name" name="name" placeholder="Enter your name" required>

<label for="email">Email:</label>
<input type="email" id="email" name="email" placeholder="Enter your email" required>

<label for="message">Message:</label>
<textarea id="message" name="message" rows="5" placeholder="Your message"></textarea>

<button type="submit">Send</button>
</form>

Input types include text, email, password, number, tel, url, date, time, color, range, file, checkbox, radio, and hidden. Each type provides appropriate validation and keyboard on mobile devices.

<input type="text" placeholder="Your name">
<input type="email" placeholder="Your email">
<input type="password" placeholder="Password">
<input type="number" min="0" max="100" step="1">
<input type="date">
<input type="checkbox" id="agree"> <label for="agree">I agree</label>
<input type="radio" name="size" value="small"> Small
<input type="radio" name="size" value="large"> Large

Select creates a dropdown menu.

<select name="country">
<option value="">Select a country</option>
<option value="us">United States</option>
<option value="uk">United Kingdom</option>
<option value="ca">Canada</option>
</select>

Tables display data in rows and columns. Use thead for headers and tbody for data.

<table>
<thead>
<tr>
<th>Name</th>
<th>Age</th>
<th>City</th>
</tr>
</thead>
<tbody>
<tr>
<td>Alice</td>
<td>30</td>
<td>New York</td>
</tr>
<tr>
<td>Bob</td>
<td>25</td>
<td>London</td>
</tr>
</tbody>
</table>

Semantic elements describe their content. Use them instead of generic divs.

<header> for the page header with logo and navigation.
<nav> for navigation links.
<main> for the primary content area.
<article> for self-contained content like a blog post.
<section> for thematic grouping of content.
<aside> for sidebar content.
<footer> for page footer with copyright and links.

<header>
<nav>
<a href="/">Home</a>
<a href="/about">About</a>
<a href="/contact">Contact</a>
</nav>
</header>
<main>
<article>
<h1>Article Title</h1>
<p>Article content...</p>
</article>
<aside>
<h2>Related Links</h2>
<ul>
<li><a href="#">Link 1</a></li>
<li><a href="#">Link 2</a></li>
</ul>
</aside>
</main>
<footer>
<p>Copyright 2025 My Website</p>
</footer>

Audio and video elements embed media.

<audio controls>
<source src="song.mp3" type="audio/mpeg">
Your browser does not support audio.
</audio>

<video controls width="640" height="360">
<source src="video.mp4" type="video/mp4">
Your browser does not support video.
</video>

The iframe element embeds external content like YouTube videos or maps.

<iframe src="https://www.youtube.com/embed/VIDEO_ID" width="560" height="315" allowfullscreen></iframe>
