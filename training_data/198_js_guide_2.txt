JavaScript DOM Manipulation. How to make web pages interactive.

The DOM, Document Object Model, is the browser's representation of the HTML page as a tree of JavaScript objects. Every HTML element becomes a node in the tree that JavaScript can read, modify, create, or delete.

Selecting elements:

document.getElementById("myId") returns one element by ID.
document.querySelector(".myClass") returns the first element matching a CSS selector.
document.querySelectorAll("p") returns all matching elements as a NodeList.
document.querySelector("nav a.active") uses any CSS selector.

Modifying content:

element.textContent = "New text"; sets plain text content.
element.innerHTML = "<strong>Bold</strong> text"; sets HTML content.
element.innerText = "Visible text"; sets text as rendered (respects CSS visibility).

Modifying attributes:

element.setAttribute("href", "https://example.com");
element.getAttribute("href"); returns the attribute value.
element.removeAttribute("disabled");
element.id = "newId";
element.src = "image.jpg";

Modifying classes:

element.classList.add("active"); adds a class.
element.classList.remove("active"); removes a class.
element.classList.toggle("active"); adds if absent, removes if present.
element.classList.contains("active"); returns true or false.
element.classList.replace("old", "new"); swaps one class for another.

Modifying styles:

element.style.color = "red";
element.style.backgroundColor = "#333"; (camelCase, not kebab-case)
element.style.display = "none"; hides element.
element.style.display = ""; restores to CSS-defined value.

For multiple styles, prefer adding a class instead of setting individual style properties.

Creating elements:

const div = document.createElement("div");
div.textContent = "Hello";
div.classList.add("card");
document.body.appendChild(div); adds to end of body.

const li = document.createElement("li");
li.textContent = "New item";
document.querySelector("ul").prepend(li); adds to beginning.

parentElement.insertBefore(newElement, referenceElement); inserts before a specific child.
element.remove(); removes element from DOM.
parent.removeChild(child); removes a specific child.

Event listeners:

element.addEventListener("click", function(event) {
  console.log("Clicked!", event.target);
});

Common events:
click - mouse click or tap
dblclick - double click
mouseover / mouseout - mouse enters/leaves
mousedown / mouseup - mouse button press/release
keydown / keyup - keyboard key press/release
input - input value changes (fires on every keystroke)
change - input value changes (fires on blur for text, immediately for select/checkbox)
submit - form submission
scroll - element or page scrolls
load - page or resource finishes loading
resize - window size changes
focus / blur - element gains/loses focus

Event delegation. Instead of adding listeners to many child elements, add one listener to the parent and check which child was clicked.

document.querySelector(".todo-list").addEventListener("click", function(event) {
  if (event.target.classList.contains("delete-btn")) {
    event.target.closest("li").remove();
  }
});

This is more efficient and works with dynamically added elements.

Preventing default behavior:

document.querySelector("form").addEventListener("submit", function(event) {
  event.preventDefault(); // prevents page reload
  const data = new FormData(event.target);
  console.log(data.get("email"));
});

link.addEventListener("click", function(event) {
  event.preventDefault(); // prevents navigation
});

Creating interactive UI patterns:

Toggle visibility:
button.addEventListener("click", () => {
  panel.classList.toggle("hidden");
});

Tab switching:
tabs.forEach(tab => {
  tab.addEventListener("click", () => {
    tabs.forEach(t => t.classList.remove("active"));
    tab.classList.add("active");
    panels.forEach(p => p.classList.remove("active"));
    document.getElementById(tab.dataset.panel).classList.add("active");
  });
});

Smooth scroll to element:
document.querySelector("#scroll-btn").addEventListener("click", () => {
  document.querySelector("#target").scrollIntoView({ behavior: "smooth" });
});

Dynamic list:
const input = document.querySelector("#new-item");
const list = document.querySelector("#item-list");
document.querySelector("#add-btn").addEventListener("click", () => {
  if (input.value.trim()) {
    const li = document.createElement("li");
    li.textContent = input.value;
    list.appendChild(li);
    input.value = "";
  }
});
